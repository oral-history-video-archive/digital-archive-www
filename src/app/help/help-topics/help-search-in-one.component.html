How to Search Within One Person's Stories Only

Searching inside a single HistoryMaker is done from the HistoryMaker's landing page. For example, suppose you are on the landing page for Barack Obama. You wish to search inside his stories for "Michelle" and can do so by using the controls on the top of the landing page.

For mobile (narrow width browsers), click the search icon (magnifying glass) and proceed to "Search this person's stories" entry area. Type the word Michelle into the entry area and either also type the Enter/Return key or tap/click the "Search" (magnifying glass) button.

For widescreen browsers, you will have a "Search this person's stories" entry box and can type there directly. Type the word Michelle into the entry area and either also type the Enter/Return key or click/tap the "Search" (magnifying glass) button.

You will get a set of stories from Barack Obama which mention "Michelle": story results for "Michelle" query within just Barack Obama.
