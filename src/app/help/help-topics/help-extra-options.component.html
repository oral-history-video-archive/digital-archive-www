Extra Options Under Development

Some features are under development. Settings for turning these on and off are at the bottom of the user settings page.

Sharing story sets can help others. Story sets might even get labeled with their author and packaged with extra detail. This type of labeled story set, named a "mixtape", was produced by four people as an example of what a "mixtape" could be. If you wish to see these four mixtapes on your home page, you need to turn on the show-mixtapes option in the user settings page.

Through inferential indexing, a story can be returned because it is tagged with certain topics, even if such topic text is not verbatim in the transcript. A topic search can then return all stories having selected tags. To always show this menu option, turn on the show-topic-search option in the user settings page.
